Frequently Asked Questions

I didn't receive the activation link. How do I proceed?

First, please check your spam. Second, if you did not receive the email, you can use the forgot password feature on the login page to get back an activation link.

Is it possible to submit DICOM images directly?

No, the currently only supported format is compressed Nifti: ".nii.gz". Please read the tutorial for help on format conversion. You can have additional help watching this video or reading this book.

The pipeline finished, but I have an issue on the obtained results. How to report a bug?

To report an issue on the platform or the pipelines, please contact: jaime.barrancohernandez@hevs.ch.

Where is my medical data stored and is it exposed?

Imaging data is temporarily stored on a physical server at the HES-SO in Sion (Valais) and is deleted immediately after the segmentation process.
