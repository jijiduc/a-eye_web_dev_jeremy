Segmentation Tool

About the segmentation tool

This is an automated pipeline for 3D segmentation of orbital structures from MRI, developed and validated on a large-scale cohort of 1,245 healthy adult subjects.

It segments nine structures of the eye (lens, globe, optic nerve, intraconal fat, extraconal fat, lateral rectus muscle, medial rectus muscle, inferior rectus muscle, superior rectus muscle) and extracts quantitative biomarkers: axial length (mm) and volumetry (mm³) per structure.

Results are provided in a .zip archive containing:

  • the original input image (_raw.nii.gz)
  • per-eye segmentation masks — left, right, and merged (.nii.gz)
  • if biomarkers are extracted: axial length visualisations (.png) and a summary table (biomarkers.csv)

Reference: A-eye paper

Important
  • T1-weighted images only. The model was trained exclusively on T1w sequences.
  • Accepted formats
    • Files for selection: .nii · .nii.gz · .zip · .7z
    • Images handled: .nii · .dcm
  • Limited to 5 cases per use.
  • Biomarkers extraction is optional. If used, the results are added to the files to download.
