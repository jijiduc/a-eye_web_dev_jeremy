Informations sur AEye

Bienvenue sur la page d’informations d'AEye. Cette plateforme vous permet de télécharger des images IRM, de les segmenter automatiquement et de télécharger les résultats. Voici un guide pour chaque étape :

Fonctionnalités

  • Upload : Utilisez le bouton "Upload" pour sélectionner et télécharger des fichiers IRM depuis votre ordinateur.
  • Segmentation : Une fois les fichiers téléchargés, cliquez sur "Segment" pour lancer le processus de segmentation automatique.
  • Download : Après la segmentation, utilisez le bouton "Download" pour récupérer les fichiers segmentés.

Astuces

Assurez-vous que les fichiers que vous téléchargez sont au format IRM compatible pour un meilleur résultat de segmentation. Si vous avez des questions, n'hésitez pas à consulter la documentation de l’application ou à contacter le support.
