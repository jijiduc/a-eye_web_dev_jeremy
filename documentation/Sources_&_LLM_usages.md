# Sources and LLM usages

## Introduction

This document lists, in chronological order, every commit that contains the use of external sources, in order to provide transparency about all external assistance used during development.


### On the generative AI tools used
During the development phase, I used the following generative AI tools, with the approval of my supervisors:

- **Gemini (Google Search)** — the Gemini assistant integrated into Google's search engine, consulted during web searches while debugging or looking for specific solutions.
- **Gemini 3.5 Flash** — the Gemini 3.5 Flash model, used for code review, debugging help, refactoring and comments checking.

No tools were used to produce the analyses, architectural choices, experimental results or conclusions, which are the result of my own work. I take accountability for the review of all content generated by AI in this work

Each use is here under specified.

## Commits

### fields sens

- `date` : commit date and time
- `message` : commit message
- `description` : summary of changes
- `source` : external source that was adapted or transposed from
- `reference` : documentation consulted
- `dependency` : third-party library/package used
- `LLM usage` : LLM contributions

    function renderMetadataTable(fields) {
        if (!fields || !Object.keys(fields).length)
            return '<em class="text-muted p-2 d-block">No metadata available.</em>';
        const rows = Object.entries(fields).map(([key, val]) => {
            const display = Array.isArray(val) ? val.join(', ') : String(val);
            return `<tr>
                <td style="white-space:nowrap; font-weight:500; width:40%;">${key}</td>
                <td style="word-break:break-all;">${display}</td>
            </tr>`;
        }).join('');
        return `<table class="table table-sm table-bordered table-striped table-hover mb-0" style="font-size:0.78em;">
            <thead class="table-dark"><tr><th>Field</th><th>Value</th></tr></thead>
            <tbody>${rows}</tbody>
        </table>`;
    }
    ```

                if (code === 0) {
                    // success is code = 0
                    self.postMessage({ id, paths: parsePaths(unzipped) });
                } else {
                    // error handling
                    self.postMessage({ id, error: '[js7z] Error during archive unzipping...' });
                }
            };
            js7z.onAbort = (cause) => {
                self.postMessage({ id, error: `[js7z] Process aborted: ${cause}` });
            };
            ```
