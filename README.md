# 🌍 A-eye: Web Platform

This submodule contains the **web-based interface** for visualizing and exploring A-eye segmentation and morphometric results.  

Features (in progress):

- Interactive 3D visualization of MR-Eye atlases  
- Quantitative morphometry dashboard  
- Segmentation result inspection and comparison
